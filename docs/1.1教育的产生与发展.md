# 1.1教育的产生与发展
`2025/12/21 22:17:50  by: 程序员·小李`

### 什么是教育

凡是有目的地增进人的知识与技能，增强人的智力与体力，影响人的思想品德的活动都是教育。


### **教育的本质属性**

有目的的培养人——`育人`

教育是人类活动特有的，动物界不存在。


### 教育的社会属性

永恒性：人类社会存在，教育就存在

历史性：因生产力发展水平、生产关系、政治制度不同，教育有不同的性质和特点

相对独立性：教育具有自身的历史继承性；教育与政治经济制度、生产力发展具有不平衡性。

生产性：教育是生产性活动

长期性：教育在人才培养周期上较长

民族性：不同国家和民族在思想、制度、方法手段等方面存在民族性差异


### 教育的基本要素

教育者、受教育者、教育中介（教育影响）

教师是学校教育者的主体，在整个教育活动中起到`主导作用`。

受教育者是教育的对象，也是`学习的主体`，具有主观能动性。


### 教育的主要形态

正规教育、非正规教育：学校教育、其他机构展开的教育

实体教育、虚拟教育：现实环境、虚拟化环境中的教育

社会教育、家庭教育、学校教育


学校教育的特点：职能专门性、组织严密性、作用的全面性、内容系统性、手段有效性、形式稳定性


### 教育的功能

个体功能：对个体发展的功能

社会功能：对社会发展的功能（社会变迁和社会流动）

正向功能：好的作用

负向功能：坏的影响

显性功能：符合预期的功能

隐性功能：非预期的功能


社会变迁功能：通过开发人的潜能，提高人的素质，引导人的社会化，影响人的社会实践，能够推动社会的变革与发展。

社会流动功能：通过教育，社会成员可以在不同岗位、层次调整与变动，重复发挥才能，实现人生价值。



### 教育的起源


**神话起源说**：朱熹，教育是`神所创造的`，体现神或天的意志，是最古老的关于教育起源的观点

**生物起源**：利托尔诺、沛西·能，教育起源于`生物本能`，是第一个关于教育起源的学说。教育起源开始从科学的角度进行解释，但没有把握住教育的目的性和社会性。

**心理起源**：孟祿，认为教育起源于`无意识模仿`。没有把握住教育的目的性、意识性和社会性。

**劳动起源**：苏联及我国学者，认为教育起源于`生产劳动`


### 教育的发展

原始社会：教育与生产劳动结合，具有自发性、全民性、广泛性、无等级性、无阶级性、原始性，机会均等。

古代社会：教育与生产劳动分离（奴隶社会开始），具有阶级性、等级性、道统性、专制性、刻板性、象征性。


### 我国古代的教育发展

夏：序，是教射的地方，校，是习武的场所

商朝：文字记载的可考学校，商代大学叫瞽宗

西周：`学在官府，政教合一`，出现礼乐射御书数——六艺

春秋战国：`私学盛行`，养士之风盛行，学术自由，出现稷下学宫

两汉：设立`太学`，`察举制`

魏晋南北朝：`九品中正制`

隋唐：`科举制`，设立`国子监`负责教育事业，六学二馆（国子学、太学、四门学、律学、书学、算学；崇文馆和弘文馆）

宋：程朱理学，`四书五经`（大学、中庸、论语、孟子；诗、书、礼、易、春秋）

明：`八股文`（没落的开始）

清：1905年废科举，开学堂


### 国外古代教育的发展

古印度：婆罗门教育，《吠陀》主要的教育内容，僧侣是唯一的教师；佛教教育，广设庙宇，形成寺院学府

古埃及：以僧为师、以吏为师

古希腊：雅典教育（德智体美和谐发展，培养政治家和商人）、斯巴达教育（培养军人）

中世纪西欧：教会教育（三科四学——文法、修辞、辩证法；算数、几何、天文、音乐）；骑士教育（骑士七技——骑马、游泳、击剑、打猎、投枪、下棋、吟诗）

文艺复兴的欧洲：提倡人文主义教育


### 近代社会的教育

1. 公立教育崛起

2. 初等教育普遍实施（德国首先普及义务教育）

3. 教育世俗化，从宗教中分离

4. 重视立法，依法治教


### 现代社会的教育

公共性：面相全体人民

生产性：教育成为经济发展的杠杆

科学性：科学教育是主要的方面

未来性：教育不仅满足于当下，还要考虑未来的社会发展要求

国际性：现代教育从态度、知识、情感、技能等方面培养受教育者从小就为一个国际化的时代做准备

终身性:终身学习，终身教育

革命性：教育处于不断变革中
