### 教育目的的概念

狭义的教育目的指处于主流地位或国家提出的教育总目的。

广义的目的是存在于人民群众心中的对受教育者的期望和要求。


教育目的由政府提出，反映社会发展的根本要求，明确而稳定，对各级各类学校的教育工作及人的成长和人才的培养具有重要的指导和规范作用。教育目的是学校教育培养人的根本质量规格。



### 教育目的的作用

定向：规定了教育和学生发展的根本方向，是学校办学的根本指导思想，学生发展总方向，是学校教育工作的起点和归宿

调控：教育目的规定了教育培养人才的基本质量规格，对学校教育的内容和形式起到选择、协作、调节、控制作用

评价：学校办学质量、学生发展质量的评判标准是教育目的。



### 教育方针

教育方针是国家在一定历史阶段有关教育的总方向和方针，是教育基本政策的总概括，是教育目的的政策性表达。

教育必须为社会主义现代化建设服务，为人民服务。

培养德智体美劳全面发展的社会主义建设者和接班人

教育必须与生产劳动和社会实践相结合

教育目的是教育方针中核心和基本内容


### 教育目的与教育方针的关系

教育目的规定的是为谁培养人、培养什么样的人。

教育方针除了上述问题，还有怎样培养人的问题，怎么办教育，办什么样的教育

### 教育目的的层次

1. 国家教育目的

2. 各级各类学校培养目标：同中有异、重点突出、特点鲜明

3. 教师的教学目标



### 教育目的与培养目标的关系

教育目的是制定培养目标的依据，同时又必须通过具体的培养目标才能得以实现。教育目的反映了培养人的方向性、共同性，培养目标体现了培养人的层次性和具体规定性；教育目的反映了一定的社会历史时期培养人的共同要求，培养目标则体现了培养人的具体要求。



### 教育目的的确认依据

1. 社会政治、经济、文化因素

2. 人的身心发展特点和需要

3. 制定者的教育理想和价值观



### 教育目的的价值取向

个人本位：教育的根本目的是人的本性和本能的高度发展

社会本位论：教育以社会的稳定和发展为宗旨

宗教本位：把人培养成虔诚的教徒

教育无目的论：杜威——无教育过程之外的“外在目的”



### 我国教育目的的理论基础

我国教育目的的理论基础是马克思关于人的全面发展学说。

1. 人的全面发展指的是体力、智力、道德发展

2. 旧式分工造成了人的片面发展

3. 机器大工业生产为人的全面发展提供基础和可能。

4. 社会主义制度是实现人的全面发展的社会条件

5. 教育与生产劳动相结合是造就全面发展的人的唯一方法


### 全面发展教育——德育

培养学生正确的人生观、世界观、价值观，使其具有良好的道德品质，正确的政治观念，形成正确的思想方法。

基本任务：

1. 培养良好的道德品质

2. 培养学生正确的政治方向

3. 培养学生正确的价值观

4. 培养学生良好、健康的心理品质

5. 培养学生良好的思想品德能力



### 全面发展教育——智育

传授系统的文化知识与技能，发展他们的智力与学习有关的非认知因素的教育

主要内容和任务：

1. 传授科学文化知识

2. 培养基本技能

3. 培养学生的智力

4. 培养良好的学习品质，热爱科学的精神


### 全面发展教育——体育

传授关于健康的知识与技能，发展体力，增强保健意识、增强体质，养成运动习惯，增强意志力

1. 指导学生锻炼身体，增强体质和健康水平

2. 让学生掌握运动锻炼的科学知识和基本技能

3. 让学生掌握身心卫生保健知识，养成良好的身心卫生保健习惯

4. 发展学生的良好品德，使学生养成文明习惯
