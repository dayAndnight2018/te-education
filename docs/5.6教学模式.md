# 5.6教学模式
`2026/01/11 22:40:16  by: 程序员·小李`

教学模式：为了实现某种教学任务建立起来的相对稳定的教学活动结构


### 程序教学——斯金纳

把学习内容`分解`成若干易于掌握的小步子，`循序渐进`地学习。

学习者可以`自主安排`学习，在教学过程中需要对学习者的学习情况进行`反馈和强化`



### 发现教学——布鲁纳

培养学生`创造性思维`为目的，以`学科的基本结构`为内容，不断`发现探究`为方法。



### 掌握学习教学——布鲁姆

使用`反馈和矫正`方法为学生提供帮助，让`绝大多数人达到学业规定要求`



### 暗示教学——洛扎诺夫

以`暗示`为手段激发学生心理潜力，提高学习效率。



### 非指导性教学——罗杰斯

以学生为中心，情感为基调，老师是促进者，学生自主学习。
